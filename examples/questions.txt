## 💡 复杂场景问题

### 场景1：计算补偿

我在一家互联网公司工作了5年，月薪12000元（含加班费），公司说要裁员，我能拿到多少补偿？

### 场景2：恶意辞退

公司以业绩不达标为由要辞退我，但其实是我拒绝了上司的不合理要求，我该怎么办？

### 场景3：被迫辞职

公司不给我安排工作，也不辞退我，就让我待岗，我可以主动辞职然后要求补偿吗？

### 场景4：合同到期

我的劳动合同下个月到期，公司说不续签了，但没有提前通知我，我能要求N+1补偿吗？

### 场景5：试用期辞退

我试用期还有2天就满了，公司突然说要辞退我，理由是不符合录用条件，但没有具体说明，这合法吗？

### 场景6：工伤后辞退

我上个月工作时受伤，现在还在恢复期，公司说要辞退我，这合法吗？我能拿到补偿吗？

### 场景7：怀孕被辞退

我怀孕3个月了，公司以业务调整为由要辞退我，这合法吗？我该怎么维权？

### 场景8：竞业协议

我离职时公司要求我签竞业限制协议，但没有说给补偿金，这合法吗？

### 场景9：社保问题

公司拖欠社保，我想辞职并要求经济补偿，可以吗？需要什么证据？

### 场景10：无劳动合同

我在公司工作了2年，但一直没签劳动合同，现在被辞退了，我能要求什么补偿？

## 📊 测试用问题集

1. 公司辞退员工应该给补偿吗？

3. 劳动仲裁需要准备什么材料？

### 中级问题（置信度应≥70%）

10. 劳动仲裁输了可以上诉吗？

### 高级问题（置信度应≥60%）

## 📝 使用方法

### 1. 单个问题测试

```bash
python -m legal_rights ask "公司辞退员工应该给补偿吗？"
```

### 2. 批量测试

```bash
python -m legal_rights.scripts.batch_test_questions
```

### 3. 交互式测试

```bash
python -m legal_rights chat

# 然后逐个输入问题
```
